* Рисуем математические формулы и получаем LaTeX (и не только):
    * [Math Webdemo](https://webdemo.myscript.com/views/math/index.html#)
* LaTeX из изображения (скриншота):

    * [GeoGebra](https://www.geogebra.org/)
